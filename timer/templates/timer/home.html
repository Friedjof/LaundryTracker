Welcome to Laundry Tracker

Laundry Tracker is a web app that helps you monitor laundry machines in a building. Users can view machine status, set timers, and mark machines as available.

With real-time updates and an intuitive interface, Laundry Tracker ensures you never wait for an available machine.

Please note: Contact your local admin to obtain a room link for marking machine status.
